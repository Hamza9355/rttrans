# Configuration EmailJS

Pour que les formulaires fonctionnent, vous devez configurer EmailJS:

## Étapes:

1. Allez sur https://www.emailjs.com/
2. Inscrivez-vous (gratuit)
3. Créez un service email
4. Créez deux templates pour:
   - Contact: `template_contact`
   - Réservation: `template_reservation`
5. Copiez votre PUBLIC_KEY
6. Remplacez "YOUR_PUBLIC_KEY" par votre clé dans:
   - `js/contact.js`
   - `js/reservation.js`
